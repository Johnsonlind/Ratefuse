# 🎬 影视多平台评分系统（Ratefuse）

## ✨ 项目简介

Ratefuse 聚合了多部影视在 豆瓣、IMDb、TMDB、Letterboxd、烂番茄、Metacritic、Trakt 等平台的评分与链接，支持用户注册登录、收藏清单、关注、通知、反馈与管理员后台。前端可导出评分卡与榜单图，并提供 PWA 与响应式布局，让你的影视评分体验更便捷、高效。

---

## 🛠 主要功能

- 🎥 **多平台评分一览与详情浏览**：一部影视多站评分一览，支持电影与剧集
- 👤 **用户认证系统**：注册、登录、JWT 鉴权、个人资料管理与密码重置
- 📂 **个性化收藏列表**：创建清单、添加/删除收藏、支持自定义排序、分享与收藏他人清单
- 🤝 **用户关注功能**：关注用户、查看关注关系
- 🔔 **站内通知**：关注用户动态、收藏清单更新及反馈状态提醒
- 🔄 **实时数据更新**：评分与榜单短期缓存，定时同步
- 🏆 **首页与公开榜单**：榜单列表及详情展示
- 🌐 **跨平台兼容**：响应式布局、PWA 支持
- 📤 **导出分享**：评分卡与榜单图 PNG 导出
- ⚙️ **管理员后台**：榜单维护、用户管理、反馈处理及平台状态监控
---

## 💻 技术栈

### 后端

- **FastAPI + Uvicorn**：API 与路由，ASGI 服务
- **SQLAlchemy + PyMySQL**：ORM 与数据库操作（表结构以 models.py 为准）
- **Redis**：异步缓存，支持评分、榜单及 TMDB 代理缓存
- **JWT（python-jose）**：登录与鉴权
- **passlib / bcrypt**：密码加密
- **httpx / aiohttp / requests**：异步 / 同步 HTTP 请求
- **Playwright**：浏览器自动化抓取
- **schedule / beautifulsoup4 / fuzzywuzzy**：榜单定时调度与内容解析

### 前端

- **TypeScript + React 18 + Vite 5**：核心开发栈
- **React Router**、**TanStack Query**、**Zustand**、**SWR**：路由与数据状态管理
- **Tailwind CSS + Headless UI + Lenis**：UI 样式与平滑滚动
- **Axios**、**react-window**、**@hello-pangea/dnd**：网络请求、长列表优化、拖拽排序
- **vite-plugin-pwa（Workbox）**：PWA 支持

### 数据与部署

- **MySQL 8+**：主库
- **TMDB / Trakt**：影视信息与图片
- **Nginx**、**Cloudflare**：生产部署与加速

---

## ⚙️ 环境要求

- **Python 3.10+**（推荐 3.12+）
- **Node.js 18+**
- **Redis 6+**
- **MySQL 8+**

## 🧩 主要功能模块

### 👤 用户与认证

- 注册、登录、登出
- JWT 鉴权（支持 Cookie 与 Bearer）
- 个人资料与邮箱重置密码流程

### 📂 收藏与清单

- 创建与管理多个收藏清单
- 清单条目自定义排序
- 收藏列表分享与收藏他人清单

- 关注 / 取关用户

- 多平台评分与链接聚合（电影、剧集及分季展示）

- 用户提交反馈与附图
- 管理端查看与处理反馈

### ⚙️ 管理后台

- 榜单录入、编辑与同步
- TMDB 搜索与媒资关联
